# Bonzomatic

Live-coding battle tool for the Revision demoscene party.

## Keys
- F5: recompile shader
- F11: hide shader overlay
- Alt-F4: exbobolate your planet

## Building
Please use Visual C++ 2010 - nothing else is needed right now.
